FIRST: Lets run some tests ...

Running the homing routine is always a good start and a good test. You can't do it to often -- feel free to run with Macro #3 as well. Watch for binding and inconsistent return to starting location.

Set Your XYZ HOME LOCATION
Clear area around the tool first!

We want to make sure all the axes are running correctly and there is no binding.

Start Motion TEST

Make Sure ALL Inputs are Working

Inputs show up in the DRO DISPLAY as shown below. There are 3 Inputs to CHECK.

Check These

  • Check GREEN BUTTON on tool (#2)
  • Check RED BUTTON (#4)
  • Check LID (#3 should come on when lid opens)
  • Check Z-Zero Clip (#1 when clip touches Zeroing-plate)

Now Ready to Do Test Cut

Before running the test file, insert the V-bit and Zero it. Use our current technique to update the setting for the height of the plate.

This routine will automatically put the test file at the back of the sample board.

Test Preparation:

  • Switch the router to ON, and set speed 4 on the router dial.
  • Dust ?
  • Close the hood
  • Now you're about to run the same test file the customers will cut in the front of the panel ...

Run the "Test-Carve"
